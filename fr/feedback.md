# Feedback et suggestions

Votre avis compte ! Aidez-nous à améliorer PitStoply en signalant les problèmes ou en proposant des idées.

---

## Signaler un bug

Vous avez rencontré un problème ? Faites-le nous savoir !

[**Signaler un bug →**](https://docs.google.com/forms/d/e/1FAIpQLSeIAZx0hIq58SKxNUPd7J0U5KGiWWeXlKlCzEbnc_XyvxMuhA/viewform)

### Informations utiles à fournir

Pour nous aider à résoudre le problème rapidement :

| Information | Exemple |
|-------------|---------|
| **Zone concernée** | Calendrier, Clients, SMS... |
| **Étapes pour reproduire** | 1. Cliquer sur X, 2. Voir l'erreur |
| **Comportement attendu** | Ce qui devrait se passer |
| **Navigateur** | Chrome, Safari, Firefox... |
| **Captures d'écran** | Si possible |

## Proposer une fonctionnalité

Une idée pour améliorer PitStoply ? Partagez-la !

[**Proposer une idée →**](https://docs.google.com/forms/d/e/1FAIpQLSdGOdc_LVTTpav78kF4To0d_792HJHRFhynXNJ6AHNCgM-xTw/viewform)

### Comment faire une bonne suggestion

Les meilleures suggestions incluent :

1. **Le problème** : Quel besoin avez-vous ?
2. **La solution** : Comment imaginez-vous la fonctionnalité ?
3. **L'importance** : À quelle fréquence l'utiliseriez-vous ?

{% hint style="info" %}
Toutes les suggestions sont lues et prises en compte pour la roadmap de PitStoply.
{% endhint %}

## Processus de traitement

### Pour les bugs

| Priorité | Délai typique |
|----------|---------------|
| Critique (bloquant) | 24-48h |
| Important | 1-2 semaines |
| Normal | Prochain sprint |

### Pour les fonctionnalités

Nous évaluons chaque suggestion selon :
- Le nombre de demandes similaires
- L'impact sur l'expérience utilisateur
- L'alignement avec la vision produit
