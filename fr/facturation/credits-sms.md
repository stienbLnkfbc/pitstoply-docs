# Crédits SMS

Les notifications SMS sont facturées via un système de crédits prépayés.

## Tarifs des packs SMS

| Pack | Prix | SMS inclus | Prix par SMS |
|------|------|------------|--------------|
| **Pack 50** | 9 CHF | 50 | 0.18 CHF |
| **Pack 150** | 19 CHF | 150 | 0.13 CHF |
| **Pack 500** | 49 CHF | 500 | 0.10 CHF |
| **Pack 1000** | 79 CHF | 1000 | 0.08 CHF |

{% hint style="tip" %}
**Recommandation** : Pour un usage régulier, le Pack 500 offre le meilleur rapport qualité-prix.
{% endhint %}

## Comment acheter des crédits

1. Allez dans **Paramètres > SMS**
2. Cliquez sur **Acheter des crédits**
3. Sélectionnez le pack souhaité
4. Procédez au paiement via Stripe

Les crédits sont ajoutés instantanément à votre solde.

## Consulter votre solde

Votre solde SMS est visible :
- Dans **Paramètres > SMS**
- Sur le Dashboard (widget notifications)

{% hint style="warning" %}
**Alerte solde bas** : Vous recevrez un email lorsque votre solde descend sous 10 SMS.
{% endhint %}

## Quand les SMS sont-ils envoyés ?

| Événement | SMS envoyé |
|-----------|------------|
| Nouvelle réservation | Confirmation (si activé) |
| 24h avant le RDV | Rappel (si activé) |
| Modification du RDV | Notification de changement |
| Annulation | Confirmation d'annulation |

## Activer/Désactiver les SMS

Dans **Paramètres > Notifications**, vous pouvez activer ou désactiver :
- Les SMS de confirmation
- Les SMS de rappel
- Les SMS de modification

{% hint style="info" %}
Même si les SMS sont activés, ils ne seront envoyés que si le client a fourni un numéro de téléphone valide.
{% endhint %}

## Historique des SMS

Consultez l'historique de tous les SMS envoyés dans **Paramètres > SMS > Historique**.

Pour chaque SMS, vous verrez :
- La date d'envoi
- Le destinataire
- Le contenu
- Le statut (envoyé, échoué)

## Questions fréquentes

<details>
<summary><strong>Les crédits expirent-ils ?</strong></summary>

Non, les crédits SMS n'expirent jamais. Ils restent sur votre compte jusqu'à utilisation.

</details>

<details>
<summary><strong>Puis-je être remboursé pour des crédits non utilisés ?</strong></summary>

Les crédits SMS ne sont pas remboursables. Nous vous recommandons d'acheter des packs adaptés à votre volume d'utilisation.

</details>

<details>
<summary><strong>Les SMS fonctionnent-ils à l'international ?</strong></summary>

Les SMS sont optimisés pour la Suisse (+41). L'envoi vers d'autres pays peut fonctionner mais n'est pas garanti.

</details>
