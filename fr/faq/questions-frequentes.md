# Questions fréquentes

## Général

<details>
<summary><strong>Qu'est-ce que PitStoply ?</strong></summary>

PitStoply est une plateforme SaaS de gestion de rendez-vous spécialement conçue pour les garages en Suisse. Elle permet à vos clients de réserver en ligne et vous aide à organiser votre planning efficacement.

</details>

<details>
<summary><strong>PitStoply est-il disponible en allemand ?</strong></summary>

Oui ! PitStoply est entièrement disponible en français et en allemand. Vous pouvez changer la langue dans **Paramètres > Général > Langue**.

</details>

<details>
<summary><strong>Puis-je essayer gratuitement ?</strong></summary>

Oui, une période d'essai est incluse lors de l'inscription. Consultez les [plans d'abonnement](../facturation/plans.md) pour plus de détails.

</details>

## Réservations

<details>
<summary><strong>Comment mes clients peuvent-ils prendre rendez-vous ?</strong></summary>

Vos clients peuvent réserver via :
1. **Votre page publique** : `pitstoply.ch/votre-slug`
2. **Le QR code client** : Chaque client a un QR code unique
3. **Vous-même** : Créez des RDV depuis votre dashboard

</details>

<details>
<summary><strong>Un client peut-il modifier ou annuler son rendez-vous ?</strong></summary>

Oui. Chaque email de confirmation contient un lien permettant au client de :
- Modifier la date/heure
- Annuler le rendez-vous

Ces options peuvent être désactivées dans vos paramètres si besoin.

</details>

<details>
<summary><strong>Comment éviter les doubles réservations ?</strong></summary>

PitStoply gère automatiquement les disponibilités. Un créneau réservé devient indisponible. De plus, vous pouvez synchroniser vos calendriers externes (Google, Outlook) pour bloquer les créneaux déjà pris.

</details>

<details>
<summary><strong>Pourquoi certains créneaux n'apparaissent pas ?</strong></summary>

Un créneau peut être masqué si :
- Il est déjà réservé
- Il tombe sur un jour/horaire fermé
- Un blocage manuel existe (congés, fermeture)
- Un événement externe bloque ce créneau (si sync calendrier activée)

</details>

## Notifications

<details>
<summary><strong>Mes clients ne reçoivent pas les emails, pourquoi ?</strong></summary>

Vérifiez :
1. L'email du client est correct (pas de faute de frappe)
2. Le client vérifie son dossier spam/indésirables
3. Les notifications email sont activées dans vos paramètres

Si le problème persiste, contactez notre support.

</details>

<details>
<summary><strong>Comment fonctionnent les crédits SMS ?</strong></summary>

Les SMS sont facturés via un système de crédits prépayés :
- Achetez un pack (50, 150, 500 ou 1000 SMS)
- Chaque SMS envoyé débite 1 crédit
- Votre solde est visible dans **Paramètres > SMS**

[Voir les tarifs SMS](../facturation/credits-sms.md)

</details>

<details>
<summary><strong>Puis-je personnaliser le contenu des messages ?</strong></summary>

Actuellement, les templates de base sont utilisés. Des options de personnalisation avancée sont prévues dans une future version.

</details>

## Configuration

<details>
<summary><strong>Quelle est la différence entre succursale et lift ?</strong></summary>

- **Succursale** = Un lieu physique (votre garage, atelier)
- **Lift** = Un poste de travail dans ce lieu (pont élévateur, baie)

Exemple : Vous avez 1 succursale avec 3 lifts = 3 rendez-vous peuvent avoir lieu en même temps.

</details>

<details>
<summary><strong>Comment configurer des horaires différents selon les jours ?</strong></summary>

Dans **Paramètres > Horaires**, vous pouvez définir pour chaque jour :
- L'heure d'ouverture
- L'heure de fermeture
- Ou marquer le jour comme fermé

Les horaires peuvent être définis au niveau de la succursale ou du lift individuel.

</details>

<details>
<summary><strong>Comment bloquer des jours de congés ?</strong></summary>

Allez dans **Paramètres > Fermetures** et ajoutez :
- Une date unique (ex: 25 décembre)
- Une période (ex: 24-31 décembre)
- Une récurrence (ex: tous les dimanches)

</details>

## Facturation

<details>
<summary><strong>Comment changer de plan d'abonnement ?</strong></summary>

Dans **Facturation > Plans**, sélectionnez le nouveau plan souhaité. Le changement prend effet :
- **Upgrade** : Immédiatement
- **Downgrade** : À la fin de la période en cours

</details>

<details>
<summary><strong>Quels moyens de paiement acceptez-vous ?</strong></summary>

Via Stripe, nous acceptons :
- Cartes de crédit (Visa, Mastercard)
- Cartes de débit
- TWINT (selon disponibilité)

</details>

<details>
<summary><strong>Où trouver mes factures ?</strong></summary>

Dans **Facturation > Gérer l'abonnement**, vous accédez au portail Stripe où toutes vos factures sont disponibles en téléchargement.

</details>

## Problèmes techniques

<details>
<summary><strong>La page se charge lentement, que faire ?</strong></summary>

1. Vérifiez votre connexion internet
2. Essayez de vider le cache de votre navigateur
3. Testez avec un autre navigateur

Si le problème persiste, [contactez le support](#contact-support).

</details>

<details>
<summary><strong>Je ne peux plus me connecter</strong></summary>

1. Vérifiez que vous utilisez le bon email
2. Utilisez "Mot de passe oublié" pour réinitialiser
3. Vérifiez vos spams pour l'email de réinitialisation

[Guide complet de dépannage connexion](../troubleshooting/connexion.md)

</details>
