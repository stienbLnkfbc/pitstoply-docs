# Glossaire

Termes utilisés dans PitStoply.

| Terme | Description |
|-------|-------------|
| **Garage** | Votre entreprise |
| **Branch** | Succursale - un lieu physique |
| **Lift** | Poste de travail (pont élévateur) |
| **Appointment** | Rendez-vous client |
| **Customer** | Client de votre garage |
| **Vehicle** | Véhicule d'un client |
| **Service** | Type de prestation |
| **Slot** | Créneau horaire disponible |
| **Booking** | Réservation en ligne |
