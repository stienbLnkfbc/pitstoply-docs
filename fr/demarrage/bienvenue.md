# Bienvenue sur PitStoply

PitStoply est votre solution tout-en-un pour la gestion de rendez-vous de votre garage. Cette documentation vous guidera pas à pas pour tirer le meilleur parti de la plateforme.

## Ce que vous pouvez faire avec PitStoply

| Fonctionnalité | Description |
|----------------|-------------|
| **Réservation en ligne** | Vos clients prennent rendez-vous 24h/24 via votre page personnalisée |
| **Calendrier intelligent** | Visualisez et gérez tous vos rendez-vous sur un calendrier optimisé |
| **Notifications automatiques** | SMS et emails envoyés automatiquement (confirmation, rappels) |
| **Multi-succursales** | Gérez plusieurs sites depuis un seul compte |
| **Multi-lifts** | Attribuez les rendez-vous à des postes de travail spécifiques |
| **Portail client QR** | Chaque client a son QR code unique pour accéder à son portail |
| **Personnalisation** | Votre logo, vos couleurs, votre domaine |
| **Rapports PDF** | Plannings journaliers et hebdomadaires exportables |
