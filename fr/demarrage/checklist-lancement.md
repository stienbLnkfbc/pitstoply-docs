# Checklist de lancement

Utilisez cette checklist pour vous assurer que tout est prêt avant d'inviter vos premiers clients.

## Configuration de base

* [ ] **Informations entreprise complètes**
  * Nom, adresse, téléphone, email renseignés
  * Slug personnalisé choisi
* [ ] **Au moins une succursale créée**
  * Nom et adresse de la succursale
  * Au moins un lift configuré
* [ ] **Horaires définis**
  * Horaires d'ouverture par succursale et/ou lift
  * Jours fermés/congés configurés (vacances, etc.)
* [ ] **Services activés**
  * Types de services sélectionnés
  * Durées ajustées selon votre organisation

## Personnalisation (optionnel mais recommandé)

* [ ] **Logo uploadé**
  * Format recommandé : PNG ou SVG
  * Dimensions idéales : 200x80 pixels minimum
* [ ] **Thème choisi**
  * Couleurs assorties à votre identité visuelle
  * Aperçu vérifié sur la page publique
* [ ] **Favicon configuré**
  * Icône affichée dans l'onglet du navigateur
* [ ] **Hero Image uploadé**
  * Image de fond d'écran sur la pagea de réservations client

## Notifications

* [ ] **Email de confirmation activé**
  * Template vérifié
  * Email de test envoyé
* [ ] **Rappel configuré**
  * Délai de rappel défini (24h recommandé)
  * Canal choisi (email et/ou SMS)
* [ ] **Crédits SMS achetés** (si SMS activés)
  * Solde suffisant pour le lancement
  * [Voir les packs SMS](../facturation/credits-sms.md)

## Vérification finale

* [ ] **Page de réservation testée**
  1. Ouvrez `pitstoply.ch/votre-slug`
  2. Sélectionnez un service
  3. Choisissez une date et un créneau
  4. Complétez avec vos infos de test
  5. Vérifiez la confirmation reçue
* [ ] **Calendrier vérifié**
  * Le RDV de test apparaît
  * Les créneaux occupés ne sont plus disponibles
* [ ] **Modification testée**
  * Cliquez sur le lien dans l'email de confirmation
  * Modifiez le créneau
  * Vérifiez la notification de modification

## Prêt pour le lancement !

{% hint style="success" %}
**Félicitations !** Si tous les points sont cochés, vous êtes prêt à partager votre lien de réservation avec vos clients.
{% endhint %}

### Comment partager votre page de réservation

| Méthode             | Description                                 |
| ------------------- | ------------------------------------------- |
| **Lien direct**     | `pitstoply.ch/votre-slug`                   |
| **QR Code**         | Générez un QR code pointant vers votre page |
| **Site web**        | Intégrez un bouton "Prendre RDV"            |
| **Signature email** | Ajoutez le lien à votre signature           |
| **Réseaux sociaux** | Partagez sur Facebook, Instagram, etc.      |

***

## Après le lancement

Une fois lancé, pensez à :

* Consulter régulièrement votre [Dashboard](../guides/calendrier.md) pour les nouveaux RDV
* Vérifier votre [solde SMS](../facturation/credits-sms.md) si activé
* Former vos employés à l'utilisation de l'interface
