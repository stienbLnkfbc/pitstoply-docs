# SMS-Guthaben

SMS-Benachrichtigungen werden über ein Prepaid-Guthaben-System abgerechnet.

## SMS-Paket-Tarife

| Paket | Preis | SMS enthalten | Preis pro SMS |
|-------|-------|---------------|---------------|
| **Paket 50** | 9 CHF | 50 | 0.18 CHF |
| **Paket 150** | 19 CHF | 150 | 0.13 CHF |
| **Paket 500** | 49 CHF | 500 | 0.10 CHF |
| **Paket 1000** | 79 CHF | 1000 | 0.08 CHF |

{% hint style="tip" %}
**Empfehlung**: Für regelmässige Nutzung bietet das Paket 500 das beste Preis-Leistungs-Verhältnis.
{% endhint %}

## Guthaben kaufen

1. Gehen Sie zu **Einstellungen > SMS**
2. Klicken Sie auf **Guthaben kaufen**
3. Wählen Sie das gewünschte Paket
4. Führen Sie die Zahlung über Stripe durch

Das Guthaben wird sofort Ihrem Konto gutgeschrieben.

## Ihr Guthaben einsehen

Ihr SMS-Guthaben ist sichtbar:
- In **Einstellungen > SMS**
- Im Dashboard (Benachrichtigungs-Widget)

{% hint style="warning" %}
**Warnung bei niedrigem Guthaben**: Sie erhalten eine E-Mail, wenn Ihr Guthaben unter 10 SMS fällt.
{% endhint %}

## Wann werden SMS gesendet?

| Ereignis | SMS gesendet |
|----------|--------------|
| Neue Buchung | Bestätigung (falls aktiviert) |
| 24h vor dem Termin | Erinnerung (falls aktiviert) |
| Terminänderung | Änderungsbenachrichtigung |
| Stornierung | Stornierungsbestätigung |

## SMS aktivieren/deaktivieren

Unter **Einstellungen > Benachrichtigungen** können Sie aktivieren oder deaktivieren:
- Bestätigungs-SMS
- Erinnerungs-SMS
- Änderungs-SMS

{% hint style="info" %}
Auch wenn SMS aktiviert sind, werden sie nur gesendet, wenn der Kunde eine gültige Telefonnummer angegeben hat.
{% endhint %}

## SMS-Verlauf

Sehen Sie den Verlauf aller gesendeten SMS unter **Einstellungen > SMS > Verlauf**.

Für jede SMS sehen Sie:
- Sendedatum
- Empfänger
- Inhalt
- Status (gesendet, fehlgeschlagen)

## Häufige Fragen

<details>
<summary><strong>Läuft das Guthaben ab?</strong></summary>

Nein, SMS-Guthaben läuft nie ab. Es bleibt auf Ihrem Konto bis zur Verwendung.

</details>

<details>
<summary><strong>Kann ich für nicht verwendetes Guthaben eine Rückerstattung erhalten?</strong></summary>

SMS-Guthaben ist nicht erstattungsfähig. Wir empfehlen, Pakete zu kaufen, die Ihrem Nutzungsvolumen entsprechen.

</details>

<details>
<summary><strong>Funktionieren SMS international?</strong></summary>

SMS sind für die Schweiz (+41) optimiert. Der Versand in andere Länder kann funktionieren, ist aber nicht garantiert.

</details>
