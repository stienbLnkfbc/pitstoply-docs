# Schliessungen und Ferien

Blockieren Sie nicht verfügbare Zeitfenster, um Buchungen während Ihrer Schliessungen zu vermeiden.

## Arten von Blockierungen

| Typ | Beispiel | Verwendung |
|-----|----------|------------|
| **Einzeldatum** | 1. August 2026 | Feiertag |
| **Zeitraum** | 24.-31. Dezember | Ferien |
| **Wiederholung** | Jeden Sonntag | Wöchentlicher Schliesstag |
| **Zeitfenster** | 15. Jan. 10-12 Uhr | Besprechung, persönlicher Termin |

---

## Blockierung erstellen

### Auf Schliessungen zugreifen

1. Gehen Sie zu **Einstellungen > Schliessungen**
2. Klicken Sie auf **Neue Schliessung**

### Punktuelle Blockierung (1 Tag)

1. Wählen Sie **Einzeldatum**
2. Wählen Sie das Datum
3. Geben Sie einen Grund an (z.B. "Nationalfeiertag")
4. Wählen Sie den Bereich (alle Hebebühnen oder eine spezifische)
5. Speichern Sie

**Beispiel: 1. August**
- Typ: Einzeldatum
- Datum: 01.08.2026
- Grund: Nationalfeiertag
- Bereich: Alle Filialen

### Zeitraum-Blockierung (mehrere Tage)

1. Wählen Sie **Zeitraum**
2. Definieren Sie das Startdatum
3. Definieren Sie das Enddatum
4. Geben Sie einen Grund an
5. Speichern Sie

**Beispiel: Weihnachtsferien**
- Typ: Zeitraum
- Beginn: 24.12.2026
- Ende: 02.01.2027
- Grund: Jahresschliessung

### Wiederkehrende Blockierung

Für regelmässige Schliessungen (jeden Sonntag, jeden Montag usw.):

1. Wählen Sie **Wiederholung**
2. Wählen Sie das Muster:
   - Jede Woche (z.B. jeden Sonntag)
   - Jeden Monat (z.B. der erste Montag des Monats)
3. Definieren Sie den Anwendungszeitraum (optional)
4. Speichern Sie

**Beispiel: Sonntag geschlossen**
- Typ: Wöchentliche Wiederholung
- Tag: Sonntag
- Grund: Wöchentliche Ruhezeit

{% hint style="tip" %}
**Tipp**: Für regelmässig geschlossene Tage (Sonntag) verwenden Sie besser die **Öffnungszeiten**, indem Sie den Tag als geschlossen markieren. Wiederkehrende Blockierungen sind nützlich für weniger standardmässige Muster.
{% endhint %}

### Spezifische Zeitfenster-Blockierung

Um nur einige Stunden zu blockieren:

1. Wählen Sie **Zeitfenster**
2. Wählen Sie das Datum
3. Definieren Sie Start- und Endzeit
4. Geben Sie einen Grund an
5. Speichern Sie

**Beispiel: Teambesprechung**
- Typ: Zeitfenster
- Datum: 15.01.2026
- Uhrzeit: 08:00 - 10:00
- Grund: Teambesprechung
- Bereich: Alle Hebebühnen

---

## Bereich der Blockierung

### Alle Filialen

Die Blockierung gilt für Ihre gesamte Werkstatt.

### Spezifische Filiale

Die Blockierung gilt nur für eine Filiale (die anderen bleiben offen).

### Spezifische Hebebühne

Die Blockierung gilt nur für eine Hebebühne (die anderen Hebebühnen bleiben verfügbar).

**Anwendungsfälle:**
- Eine Hebebühne in Wartung
- Ein abwesender Mitarbeiter
- Ein vorübergehend unbenutzbar Bereich

---

## Visualisierung der Blockierungen

### In der Liste

**Einstellungen > Schliessungen** zeigt alle Ihre Blockierungen:
- Aktiv (zukünftiges Datum)
- Vergangen (Verlauf)

### Im Kalender

Blockierungen erscheinen:
- In dunkelgrau mit Schraffur
- Mit dem Label des Grundes

---

## Blockierung bearbeiten

1. Öffnen Sie die Blockierung in der Liste
2. Ändern Sie die Informationen
3. Speichern Sie

{% hint style="info" %}
Das Ändern einer Blockierung betrifft keine bereits gebuchten Termine ausserhalb des blockierten Zeitraums.
{% endhint %}

---

## Blockierung löschen

1. Öffnen Sie die Blockierung
2. Klicken Sie auf **Löschen**
3. Bestätigen Sie

Das Zeitfenster wird wieder für Buchungen verfügbar.

## Auswirkung auf Buchungen

### Neue Termine

Blockierte Zeitfenster sind für Kunden **unsichtbar**. Sie können während dieser Zeiten nicht buchen.

### Bestehende Termine

{% hint style="danger" %}
**Wichtig**: Wenn Sie eine Blockierung für einen Zeitraum erstellen, in dem bereits Termine existieren, werden diese Termine **nicht automatisch storniert**. Sie müssen sie manuell verwalten.
{% endhint %}

### Empfohlene Überprüfung

Bevor Sie eine Blockierung erstellen:
1. Überprüfen Sie den Kalender für den betreffenden Zeitraum
2. Kontaktieren Sie die Kunden, wenn Termine geplant sind
3. Stornieren oder verschieben Sie die Termine bei Bedarf
4. Erstellen Sie dann die Blockierung

---

## Bewährte Praktiken

### Im Voraus planen

- Erstellen Sie Ihre Ferienblockierungen mehrere Wochen im Voraus
- Dies vermeidet Last-Minute-Buchungen, die verschoben werden müssen

### Gründe dokumentieren

- Verwenden Sie klare Gründe ("Ferien", "Wartung")
- Dies hilft Ihnen, später zu verstehen, warum ein Tag blockiert war

### Regelmässig überprüfen

- Löschen Sie veraltete Blockierungen
- Aktualisieren Sie Wiederholungen, wenn sich Ihre Gewohnheiten ändern

## Häufige Fragen

<details>
<summary><strong>Kann ich ein bereits gebuchtes Zeitfenster blockieren?</strong></summary>

Ja, aber der bestehende Termin wird nicht automatisch storniert. Sie müssen ihn manuell verwalten (stornieren oder verschieben).

</details>

<details>
<summary><strong>Wie blockiere ich schnell mehrere nicht aufeinanderfolgende Tage?</strong></summary>

Erstellen Sie eine separate Blockierung für jedes Datum. Es ist nicht möglich, mehrere nicht aufeinanderfolgende Daten auf einmal auszuwählen.

</details>

<details>
<summary><strong>Beeinflussen Blockierungen Ereignisse des externen Kalenders?</strong></summary>

Nein, PitStoply-Blockierungen sind unabhängig von Ihrem externen Kalender. Sie addieren sich: Wenn ein Zeitfenster in dem einen ODER anderen blockiert ist, ist es nicht verfügbar.

</details>

<details>
<summary><strong>Kann ich den Verlauf vergangener Blockierungen sehen?</strong></summary>

Ja, unter **Einstellungen > Schliessungen** können Sie vergangene und aktive Blockierungen sehen. Vergangene Blockierungen werden als Referenz aufbewahrt.

</details>
