# Feedback und Vorschläge

Ihre Meinung zählt! Helfen Sie uns, PitStoply zu verbessern, indem Sie Probleme melden oder Ideen vorschlagen.

## Einen Bug melden

Sie haben ein Problem festgestellt? Lassen Sie es uns wissen!

[**Einen Bug melden →**](https://docs.google.com/forms/d/e/1FAIpQLSeIAZx0hIq58SKxNUPd7J0U5KGiWWeXlKlCzEbnc_XyvxMuhA/viewform)

### Nützliche Informationen

Um uns zu helfen, das Problem schnell zu lösen:

| Information | Beispiel |
|-------------|----------|
| **Betroffener Bereich** | Kalender, Kunden, SMS... |
| **Schritte zur Reproduktion** | 1. Auf X klicken, 2. Fehler sehen |
| **Erwartetes Verhalten** | Was passieren sollte |
| **Browser** | Chrome, Safari, Firefox... |
| **Screenshots** | Wenn möglich |

## Eine Funktion vorschlagen

Eine Idee zur Verbesserung von PitStoply? Teilen Sie sie!

[**Eine Idee vorschlagen →**](https://docs.google.com/forms/d/e/1FAIpQLSdGOdc_LVTTpav78kF4To0d_792HJHRFhynXNJ6AHNCgM-xTw/viewform)

### Wie man einen guten Vorschlag macht

Die besten Vorschläge beinhalten:

1. **Das Problem**: Welchen Bedarf haben Sie?
2. **Die Lösung**: Wie stellen Sie sich die Funktion vor?
3. **Die Wichtigkeit**: Wie oft würden Sie sie nutzen?

{% hint style="info" %}
Alle Vorschläge werden gelesen und für die PitStoply-Roadmap berücksichtigt.
{% endhint %}

## Bearbeitungsprozess

### Für Bugs

| Priorität | Typische Frist |
|-----------|----------------|
| Kritisch (blockierend) | 24-48h |
| Wichtig | 1-2 Wochen |
| Normal | Nächster Sprint |

### Für Funktionen

Wir bewerten jeden Vorschlag nach:
- Anzahl ähnlicher Anfragen
- Auswirkung auf die Benutzererfahrung
- Ausrichtung auf die Produktvision
