# Registrierung und erster Zugang

Erstellen Sie Ihr PitStoply-Konto in wenigen Minuten und beginnen Sie mit der Terminverwaltung.

## Konto erstellen

### Schritt 1: Zur Registrierung gehen

1. Besuchen Sie [pitstoply.ch](https://pitstoply.ch)
2. Klicken Sie auf **Kostenlos starten** oder **Registrieren**

### Schritt 2: Formular ausfüllen

| Feld | Beschreibung |
|------|--------------|
| **Name** | Ihr vollständiger Name |
| **E-Mail** | Berufliche E-Mail-Adresse (wird Ihr Benutzername) |
| **Passwort** | Mindestens 8 Zeichen |
| **Werkstattname** | Der Name Ihres Unternehmens |

{% hint style="info" %}
Verwenden Sie eine E-Mail-Adresse, auf die Sie Zugriff haben - Sie erhalten eine Bestätigungs-E-Mail.
{% endhint %}

### Schritt 3: E-Mail bestätigen

1. Überprüfen Sie Ihren Posteingang
2. Öffnen Sie die E-Mail von PitStoply
3. Klicken Sie auf den Bestätigungslink

{% hint style="warning" %}
**Keine E-Mail erhalten?** Überprüfen Sie Ihren Spam-Ordner oder [fordern Sie eine neue an](#bestaetigungs-email-erneut-senden).
{% endhint %}

### Schritt 4: Plan auswählen

Nach der Bestätigung wählen Sie Ihren Abonnementplan:

| Plan | Preis | Ideal für |
|------|-------|-----------|
| **Independent** | 290 CHF/Jahr | Werkstatt mit 1 Hebebühne |
| **Multi-Lift** | 490 CHF/Jahr | Werkstatt mit mehreren Hebebühnen |
| **Network** | Auf Anfrage | Werkstattnetzwerk |

Je nach aktuellem Angebot kann eine Testphase verfügbar sein.

---

## Erste Anmeldung

### Auf Ihr Konto zugreifen

1. Gehen Sie zu [app.pitstoply.ch](https://app.pitstoply.ch)
2. Geben Sie Ihre E-Mail und Ihr Passwort ein
3. Klicken Sie auf **Anmelden**

### Konfigurationsassistent

Bei Ihrer ersten Anmeldung führt Sie ein Assistent durch:

1. **Unternehmensinformationen** - Name, Adresse, Kontakt
2. **Filialen und Hebebühnen** - Ihre Standorte und Arbeitsplätze
3. **Dienstleistungen** - Angebotene Servicetypen
4. **Benachrichtigungen** - E-Mail/SMS-Konfiguration

[Siehe Leitfaden zur Erstkonfiguration](erstkonfiguration.md)

---

## Bestätigungs-E-Mail erneut senden {#bestaetigungs-email-erneut-senden}

Wenn Sie die E-Mail nicht erhalten haben:

1. Gehen Sie zurück zur Anmeldeseite
2. Klicken Sie auf **Bestätigungs-E-Mail erneut senden**
3. Geben Sie Ihre E-Mail-Adresse ein
4. Überprüfen Sie Ihren Posteingang (und Spam)

---

## Passwort vergessen

1. Klicken Sie auf der Anmeldeseite auf **Passwort vergessen**
2. Geben Sie Ihre E-Mail-Adresse ein
3. Überprüfen Sie Ihren Posteingang
4. Klicken Sie auf den erhaltenen Link
5. Legen Sie ein neues Passwort fest

{% hint style="info" %}
Der Zurücksetzungslink läuft nach 60 Minuten ab.
{% endhint %}

## Häufige Fragen

<details>
<summary><strong>Kann ich meine E-Mail nach der Registrierung ändern?</strong></summary>

Ja, unter **Einstellungen > Profil** können Sie Ihre E-Mail-Adresse ändern. Eine Bestätigung wird an die neue Adresse gesendet.

</details>

<details>
<summary><strong>Kann ich mehrere Konten für dieselbe Werkstatt haben?</strong></summary>

Es gibt nur ein "Inhaber"-Konto pro Werkstatt. Sie können **Mitarbeiter-Benutzer** unter **Einstellungen > Benutzer** hinzufügen.

</details>

<details>
<summary><strong>Wie lösche ich mein Konto?</strong></summary>

Besuchen Sie unsere [Feedback-Seite](../feedback.md), um die Löschung Ihres Kontos und Ihrer Daten über das Kontaktformular zu beantragen.

</details>
