# Start-Checkliste

Verwenden Sie diese Checkliste, um sicherzustellen, dass alles bereit ist, bevor Sie Ihre ersten Kunden einladen.

## Grundkonfiguration

* [ ] **Unternehmensinformationen vollständig**
  * Name, Adresse, Telefon, E-Mail ausgefüllt
  * Personalisierter Slug gewählt
* [ ] **Mindestens eine Filiale erstellt**
  * Name und Adresse der Filiale
  * Mindestens eine Hebebühne konfiguriert
* [ ] **Öffnungszeiten definiert**
  * Öffnungszeiten pro Filiale und/oder Hebebühne
  * Geschlossene Tage/Urlaub konfiguriert (Ferien usw.)
* [ ] **Dienstleistungen aktiviert**
  * Servicetypen ausgewählt
  * Dauer nach Ihrer Organisation angepasst

## Personalisierung (optional aber empfohlen)

* [ ] **Logo hochgeladen**
  * Empfohlenes Format: PNG oder SVG
  * Ideale Abmessungen: mindestens 200x80 Pixel
* [ ] **Theme gewählt**
  * Farben passend zu Ihrer visuellen Identität
  * Vorschau auf der öffentlichen Seite überprüft
* [ ] **Favicon konfiguriert**
  * Symbol, das im Browser-Tab angezeigt wird
* [ ] **Hero-Bild hochgeladen**
  * Hintergrundbild auf der Kundenbuchungsseite

## Benachrichtigungen

* [ ] **Bestätigungs-E-Mail aktiviert**
  * Vorlage überprüft
  * Test-E-Mail gesendet
* [ ] **Erinnerung konfiguriert**
  * Erinnerungsvorlauf festgelegt (24h empfohlen)
  * Kanal gewählt (E-Mail und/oder SMS)
* [ ] **SMS-Guthaben gekauft** (wenn SMS aktiviert)
  * Ausreichendes Guthaben für den Start
  * [Siehe SMS-Pakete](../abrechnung/sms-guthaben.md)

## Abschlussprüfung

* [ ] **Buchungsseite getestet**
  1. Öffnen Sie `pitstoply.ch/ihr-slug`
  2. Wählen Sie einen Service
  3. Wählen Sie ein Datum und ein Zeitfenster
  4. Vervollständigen Sie mit Ihren Testdaten
  5. Überprüfen Sie die erhaltene Bestätigung
* [ ] **Kalender überprüft**
  * Der Testtermin erscheint
  * Belegte Zeitfenster sind nicht mehr verfügbar
* [ ] **Änderung getestet**
  * Klicken Sie auf den Link in der Bestätigungs-E-Mail
  * Ändern Sie das Zeitfenster
  * Überprüfen Sie die Änderungsbenachrichtigung

## Bereit für den Start!

{% hint style="success" %}
**Herzlichen Glückwunsch!** Wenn alle Punkte abgehakt sind, können Sie Ihren Buchungslink mit Ihren Kunden teilen.
{% endhint %}

### Wie Sie Ihre Buchungsseite teilen

| Methode | Beschreibung |
|---------|--------------|
| **Direktlink** | `pitstoply.ch/ihr-slug` |
| **QR-Code** | Generieren Sie einen QR-Code, der auf Ihre Seite verweist |
| **Website** | Integrieren Sie einen "Termin buchen"-Button |
| **E-Mail-Signatur** | Fügen Sie den Link zu Ihrer Signatur hinzu |
| **Soziale Medien** | Teilen Sie auf Facebook, Instagram usw. |

---

## Nach dem Start

Nach dem Start denken Sie daran:

* Regelmässig Ihr [Dashboard](../anleitungen/kalender.md) auf neue Termine zu überprüfen
* Ihr [SMS-Guthaben](../abrechnung/sms-guthaben.md) zu überprüfen, wenn aktiviert
* Ihre Mitarbeiter in der Nutzung der Oberfläche zu schulen
